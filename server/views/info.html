**Features**

- **Easy Task Entry**: Provides a user-friendly interface to quickly add tasks. - **Task Editing**: You can edit tasks to suit your needs at any time. - **Task Deletion**: Easily delete tasks when you've completed them or no longer need to do them.
**How to Use**
To start using To Do App, you can follow these steps: 1. **Add Tasks**: Enter your task in the text box on the main screen and click "Add" to add a new task. 2. **Edit Tasks**: Click on an existing task to edit it and make changes. 3. **Delete Tasks**: To delete a task when it's completed or no longer needed, simply click the "Delete" button next to it.
